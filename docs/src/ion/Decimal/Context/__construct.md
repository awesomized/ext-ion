# void ion\Decimal\Context::__construct(int $digits, int $eMax, int $eMin, ion\Decimal\Context\Rounding|int $round, bool $clamp)

Create a new decimal context.






## Params:

* int $digits  
   Maximum digits.
* int $eMax  
   Maximum exponent.
* int $eMin  
   Minimum exponent.
* ion\Decimal\Context\Rounding|int $round  
   Rounding mode.
* bool $clamp  
   Whether to clamp.
